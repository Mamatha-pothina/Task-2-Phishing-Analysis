# Task 2: Phishing Email Analysis

## Objective
To analyze a phishing email and identify suspicious elements.

## Tools Used
- Notepad (Text Editor)

## Sample Email
A fake PayPal email claiming account suspension and asking for verification.

## Phishing Indicators Identified

1. **Spoofed Email Address**
   - The sender email "support@paypa1.com" is fake (uses '1' instead of 'l').

2. **Urgent Language**
   - Words like "urgent" and "immediately" are used to create panic.

3. **Suspicious Link**
   - The link provided is not an official PayPal domain.

4. **Generic Greeting**
   - Uses "Dear Customer" instead of addressing the user by name.

5. **Threat Message**
   - Mentions account suspension to pressure the user.

## Conclusion
The email is a phishing attempt that uses social engineering techniques to trick users into revealing sensitive information.
